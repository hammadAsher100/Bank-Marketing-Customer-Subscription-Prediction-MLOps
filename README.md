## 📊 Project Overview

> **Predict whether a bank client will subscribe to a term deposit** based on telemarketing campaign data, economic indicators, and client information.

<div align="center">

| Aspect | Detail |
|:------:|:------:|
| 🎯 **ML Task** | Binary Classification |
| 📈 **Dataset Size** | 45,211 records |
| 🔢 **Features** | 16 input variables |
| ⚖️ **Class Balance** | 11% Positive / 89% Negative |
| 🧹 **Missing Values** | None |
| 🏆 **Primary Metric** | AUC-ROC, F1 Score |
